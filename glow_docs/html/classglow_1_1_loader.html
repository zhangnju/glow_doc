Get the description of the model inputs provided through the command line interface to the Loader by providing separately the names in the vector inputNames and the input types in the vector inputTypes.

Parse the 'modelInputsOpt' option and get the model input names and types. The expected format is one of the following:

  • <name> (default type is 'float', default shape is '[1]')
  • <name>,<type>,<shape> for non-quantized types.
  • <name>,<type>,<scale>,<offset>,<shape> for quantized types.
