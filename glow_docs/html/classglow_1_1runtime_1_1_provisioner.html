Detailed Description

The Provisioner is responsible for assigning networks to an actual device. It also compiles the networks before passing the compiled functions to the device.

Traverses the DAG networks and:

  1. Retrieves each node's Function from the provided module.
  2. Compiles it using the provided CompilationContext cctx.
  3. Assigns a device and calls addNetwork on the chosen device(s).

    a Error indicating if the operation was a success.
