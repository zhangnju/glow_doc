Detailed Description

This is a class containing a common logic for the generation of the LLVM IR from an IRFunction. The primary clients of this class are JITs and bundlers.

Emit IR for the data parallel instruction I which is invoked inside the stacked kernel. The current loop count is described by loopCount. The bufferToArgNum map can be used to find the required buffers, which are provided as arguments to the stacked kernel.
