Detailed Description

Represents a single RNN model: encoder combined with decoder. Stores vocabulary, compiled Graph (ready to be executed), and few references to input/output Variables.

Model part representing Encoder. Remembers input sentence into hidden layer. input is Variable representing the sentence. seqLength is Variable representing the length of sentence. encoderHiddenOutput saves resulting hidden layer.

Translation has 2 stages: 1) Input sentence is fed into Encoder word by word. 2) "Memory" of Encoder is written into memory of Decoder. Now Decoder streams resulting translation word by word.
