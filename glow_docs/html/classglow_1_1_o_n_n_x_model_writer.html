Creates an ONNX model writer to serialize F graph into file modelFilename, writing irVersion and opsetVersion. If errPtr is not null then if an error occurs it will get assigned there otherwise if an error occurs it will abort. It also supports serialization with text format or binary format depending on textMode. If zipMode is true, it will save weights into individual TensorProto file along with the model file and package them into a zip file.
