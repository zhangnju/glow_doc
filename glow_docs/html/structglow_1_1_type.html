◆ strides_

size_t glow::Type::strides_[max_tensor_dimensions]
Initial value:
= {
0,
}

Contains the strides for each dimension (in elements). The order should be the same as in sizes_. In more details, suppose that the tensor is laid out flat in memory, and some dimensions are aligned. strides_[i] is the number of elements that needs to be skipped in order to reach the next plane in the i-th dimension. For example, if the tensor has dimensions [3, 5, 10] and alignments [3, 32, 1], the strides will be [162, 32, 1].
