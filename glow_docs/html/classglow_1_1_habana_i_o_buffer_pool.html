Detailed Description

A pool of HabanaIOBuffer objects that threads must "check out" before they execute a HabanaFunction and "check in" after execution is complete. To enable concurrency, this class can allocate several times the required memory for all input/output Placeholders to allow multiple threads to execute on the device at once.
