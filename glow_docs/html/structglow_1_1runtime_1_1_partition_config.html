◆ nodeToPartition

llvm::StringMap<size_t> glow::runtime::PartitionConfig::nodeToPartition

The mapping between nodes' name to Partition ids. Assume there are n nodes and m partitions. We have 2 types of valid mapping: 1. all nodes are mapped to a partition. 2. For i-th (0 <= i < m) partition, the nodes mapped to this partition id are not in this map, and the nodes mapped to other partitions ids must be in this map. The node's name should be the name in Glow function and may be different from the original name from models. Since Glow will mangle names to make them unique.
