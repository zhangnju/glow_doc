Detailed Description

This class implements the high-level APIs used to convert a function from one type to another. The actual conversions must be implemented by derived classes.

◆ convert()

void FunctionConverter::convert ( )

Convert F according to ::getTargetTypeForOutput and ::getTargetTypeForInput.

The high level algorithm looks like:

for each node in function:
insert conversions for the inputs of node
update the inputs of node to use the results of the conversions
mutate the type of the outputs of node
insert conversions for the outputs of node
morph node
postProcessing node
cleanUp

◆ getTargetTypeForInput()

TypeRef FunctionConverter::getTargetTypeForInput ( const Node &  use,
unsigned  idx 
) const
protectedvirtual
Returns
the type that the input operand described by idx-th input of use needs to have at the end of the conversion procedure. In other words, this is the type this value will have at the end of ::convert. E.g., let say we want to convert: 
/// res = matmul float A, B
///
into 
/// res = matmul fp16 A, B
///
The target type for A (i.e., (matmul, 0)) is fp16.

Using this information, the conversion procedure will insert a conversion of (node, idx) from its current type to the returned type. 

/// convertedA = convert A's current type A to returned type
/// res = matmul fp16 convertedA, B
/// 
If nullptr is returned or the returned type is identical to the current
type of the related value, no conversion will be inserted by the
conversion procedure.
