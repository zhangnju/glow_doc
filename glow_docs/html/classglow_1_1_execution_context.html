Detailed Description

The runtime context for a single execution (Inferance or Training) in the the Glow Execution Engine or HostManager. This class includes the mapping between Input/Output Placeholders and the materialized Tensors used for this run, the set of Device specific details required to execute the function, and stores TraceEvents that were generated as a result of the run.
