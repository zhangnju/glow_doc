◆ assertAllNodesQuantized

bool glow::quantization::QuantizationConfiguration::assertAllNodesQuantized {false}

If true, the quantizer will abort when encountering a node that it would like to quantize but the backend cannot support. Note that node kinds in doNotQuantizeKinds will skip this check and not cause an abort.
