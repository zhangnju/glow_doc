◆ precisionModeKindSet

KindSet glow::PrecisionConfiguration::precisionModeKindSet

Used during Quantization and convertToFP16 to keep the original precision of specific node kinds (i.e. quantization/FP16 conversion would be skipped for any node kinds found here). Used during profiling to prevent nodes from being lowered before instrumenting the graph (e.g. do not lower group convolutions for profiling; see -do-not-lower-nodes-for-profiling in docs/Quantization.md).
