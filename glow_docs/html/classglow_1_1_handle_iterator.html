Detailed Description

template<class ElemTy, bool IsConst>
class glow::HandleIterator< ElemTy, IsConst >

A class that provides ability to iterate over a Handle<ElemTy>. Since it's common to have both mutating and const iterators, this class has template parameter IsConst, which is true to create const_iterator and false otherwise.
