Detailed Description

This class provides a mapping between some graph nodes, which are a symbolic representation of some computation, and concrete tensors that represent the inputs and outputs to the graph. The PlaceholderBindings owns the tensors and the graph uses these values as runtime. This is useful for the multi-threaded execution of code, where each thread has a different execution context. The difference between this class and a regular map is that the PlaceholderBindings owns the Tensors (not only the pointers) and manages their lifetime.
