Detailed Description

This is a scheduler based on the generalized the paper "Generalizations of the Sethi-Ullman algorithm for register allocation" by Andrew W. Appel and Kenneth J. Supowit. http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.54.319&rep=rep1&type=pdf

The idea is to give more priority and schedule earlier those child nodes that free more memory after their computation.
