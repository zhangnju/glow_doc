Detailed Description

template<bool is_const_iter = false>
class glow::NodeValueIteratorImpl< is_const_iter >

Helper class to walk through the specific uses of a NodeValue. This class is built on top of the regular users-list (Node::getUsers) but filters out the uses that don't affect the desired NodeValue.
