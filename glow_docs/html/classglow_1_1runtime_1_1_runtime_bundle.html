Detailed Description

Contains the information needed to be passed forward from compile time to runtime. In order to allocate and initialize memory.

◆ collectConstants()

void glow::runtime::RuntimeBundle::collectConstants ( const IRFunction *  F)

At compile time condense constants to a single block of memory. This allows the graph to go away after compile time. Allocates a block of memory of size constantMaxSize then walks the given function F and and copies weights to their address as specified by offsets contained in symbolTable_.

runtime::RuntimeBundle runtime::RuntimeBundle::create ( const Function &  F)
static

Build a runtime symbol table from a Function. Computes Constant and Placeholder sizes, but not Activations, since Functions are unserialized. Only use this method to generate bundles for backends that do not use Glow's IR.
