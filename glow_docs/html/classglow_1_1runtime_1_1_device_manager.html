Execute the named Function in an already provided network on the device. functionName must match the name of a function already added. The ExecutionContext's PlaceholderBindings should have all Placeholders allocated. resultCB will be called with the ExecutionContext containing output tensors filled, and any generated TraceEvents.

Implemented in glow::runtime::HabanaDeviceManager, glow::runtime::DummyDeviceManager, glow::runtime::NNPIDeviceManager, and glow::runtime::QueueBackedDeviceManager.
