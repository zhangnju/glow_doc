Detailed Description

A class that contains an openCL device buffer. It frees the buffer when it is destroyed. Can be extended to store multiple buffers and rotate through them. Also tracks number of functions using this buffer. Since adds/evicts are serialized by the DeviceManager this does not need multithreading protection.
