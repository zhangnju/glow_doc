Creates a ONNX model loader to build F. Loads the ONNIXFI model from memory of modelSize size, and weightsCount, and onnxTensorDescriptorV1 correspondent descriptors. Converts inputs into placeholder if requested loadInputsAsPlaceholders. Reports success/failure through optional parameter errPtr.

Loads the ONNX model that's represented by a model description file, serialized in modelDescFilename and populates the network into F. The types in types match the list of names tensorNames and used as inputs to the network. If names and types are empty loader fills inputs automatically. If errPtr is not null then if an error occurs it will get assigned there otherwise if an error occurs it will abort.
