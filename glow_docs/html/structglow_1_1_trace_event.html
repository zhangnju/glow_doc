Detailed Description

An individual tracing event, such as the begin or end of an instruction. Designed to match the Google Trace Event Format for Chrome: https://docs.google.com/document/d/1CvAClvFfyA5R-PhYUmn5OOQtYMH4h6I0nSsKchNAySU
